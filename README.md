## Background

- To solve the problem of being banned from uploading files with sensitive text in the file name.
- The project is based on tauri + vue3 + typescript + tailwind.


## Install

### install dependencies
```sh
npm install
```

### dev
```sh
npm run tauri dev
```

### build
```
npm run tauri build
```
